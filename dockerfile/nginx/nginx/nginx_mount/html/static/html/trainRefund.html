退票规则


时间规则:

开车前15天（不含）以上退票的，不收取退票费。

开车前48小时以上,15天以内的按票价5%收取退票费。

开车前24小时以上、不足48小时的按票价10%收取退票费。

开车前不足24小时的按票价20%收取退票费。

开车前48小时〜15天期间内，改签或变更到站至距开车15天以上的其他列车，又在距开车15天前退票的，仍核收5%的退票费。

注：1.以上线上退票只适用于未取出纸质车票且不晚于开车前30分钟的车票。

退票费最低按2元计收。

注：1.以上线上退票只适用于未取出纸质车票且不晚于开车前30分钟的车票。

改签后的车票乘车日期在春运期间的，退票时一律按开车时间前不足24小时标准核收退票费。

对于停运车次，退票是否收取手续费的问题，一律根据12306发布的公告。
