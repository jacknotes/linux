改签规则


时间规则:

火车票改签最晚应不晚于列车发车日期当日的24:00。(开车前48小时(不含)以上，可改签预售期内的其他车次;开车前48小时以内，可改签开车前的其他车次，也可改签开车后至票面日期当日24:00之间的其他列车，不办理票面日期次日及以后的改签;开车之后，旅客仍可改签当日其他列车，但只能在票面发站办理改签)。

在12306网站上改签应不晚于列车发车前30分钟。


收费规则:

火车票改签是不收手续费的。

改签后的 新车票票价高于原车票的，补收差额;新车票票价低于原车票的，退还差额。


改签后退票规则：

火车票按梯次收取退票手续费。

发车前15天（不含）以上退票，无退票费。

发车前49小时以上，收5%退票费。

发车前25小时～49小时，收10%退票费。

发车前25小时以内，收20%退票费。

手续费不足2元按2元收取，最终退款以铁路局实退为准。

保险订单在退票后会全额退还保险。

改签至春运期间的车票申请退票收取20%手续费。


改签说明：

开车前48小时〜15天期间内，改签至距开车15天以上的其他列车，又在距开车15天前退票的，仍核收5%的退票费。

改签后的车票乘车日期在春运期间的，退票时一律按开车时间前不足24小时标准核收退票费。


改签注意事项：

每张车票只有一次改签的机会。

未取票且不晚于列车发车前30分钟的情况下，旅客可以在12306网站自己改签。

服务时间为7：00~23：00，改签需在此时间段内操作。
